# MockLocation
Mock Location is a powerful and easy-to-use mock location app that lets you set your GPS to anywhere in the India with a single tap. Perfect for protecting your privacy on social media, testing location-based apps, playing AR games, or planning your next trip. Take full control of your device's location data.

NOTE : To run this project, you need to add your Google Maps API key to the local.properties file.

Step-by-Step Guide to Using a Mock Location App
Part 1: Initial Setup (Enable Developer Options)
This is a one-time setup required for all mock location apps on Android.

Go to Settings on your phone.

Scroll down and tap About Phone (or About Device).

Find Build Number. Keep scrolling; it's usually at the very bottom.

Tap Build Number 7 times.

You will see a countdown message like "You are now 4 steps away from being a developer."

After the 7th tap, you'll see a message: "You are now a developer!"

Go back to the main Settings menu. You will now see a new option called Developer Options (or System > Developer Options).

Part 2: Configure Developer Options for Mock Location
Open Developer Options.

Scroll down to the Debugging section.

Find and select Select Mock Location App. (On some older Android versions, it might be called Allow Mock Locations, which you would simply toggle on).

A list will pop up. Select your mock location app (e.g., "LocationFlex").

You have now authorized the app to change your device's GPS.
